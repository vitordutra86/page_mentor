
# Expert Interleaving

## Descrição

Combina diversas perspectivas de especialistas dentro de uma tarefas, reproduzindo respostas mais robustas e multifacetadas. É a aplicação de diversos papéis numa mesma interação. Muito parecida com o Diálogo Socrático, é uma técnica que busca obter pontos de vista distintos sobre uma mesma questão. Ótima técnica para gerar empatia.

## Templates (Prompts)

```
Técnica: Expert Interleaving
### Input ###
Atue alternando três papéis diferentes para construir a resposta:
### Perspectivas ###
- Perspectiva técnica/jurídica: [Explicação rigorosa e fundamentada]
- Perspectiva didática: [Exemplificação prática e simplificada]
- Perspectiva comunicacional: [Conteúdo adaptado para fácil compreensão e divulgação]
### Orientações ao usuário ###
* Altere as perspectivas quando entender necessário.
### Output ###
Integração das perspectivas em resposta final:
```
